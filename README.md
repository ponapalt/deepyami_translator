# DeepYami翻訳アプリ

LangChainと複数のLLMモデルを使用した、DeepL風の翻訳アプリケーション

## 機能

- 🌐 多言語対応（日本語、中国語簡体字、中国語繁体字、韓国語、英語）
- 🤖 複数のLLMモデルに対応
  - OpenAI GPT-4.1
  - Anthropic Claude Sonnet 4.5
  - Google Gemini 2.5 Pro
- 📝 メモ帳風のシンプルなUI
- 💾 ファイルの開く・保存機能
- ⚙️ 簡単な設定管理

## 必要要件

- Python 3.8以上
- Windows OS（start.batを使用する場合）
- 以下のいずれかのAPIキー：
  - OpenAI API Key
  - Anthropic API Key
  - Google API Key

## セットアップ（Windows）

1. リポジトリをクローンまたはダウンロード
2. `start.bat` をダブルクリック
3. 初回起動時に仮想環境と依存関係が自動的にインストールされます

## 使い方

### 初回起動

1. アプリケーションを起動
2. 「設定」メニューから「API設定」を選択
3. 使用したいLLMモデルを選択
4. 対応するAPIキーを入力
5. 「保存」をクリック

### 翻訳

1. 左側の言語選択で翻訳元の言語を選択
2. 右側の言語選択で翻訳先の言語を選択
3. 左側のテキストエリアに翻訳したいテキストを入力
4. 「翻訳」ボタンをクリック
5. 右側のテキストエリアに翻訳結果が表示されます

### ファイル操作

- **新規作成**: ファイル → 新規作成
- **開く**: ファイル → 開く
- **保存**: ファイル → 保存
- **名前を付けて保存**: ファイル → 名前を付けて保存

## APIキーの取得方法

### OpenAI API Key
1. https://platform.openai.com/ にアクセス
2. アカウントを作成/ログイン
3. API Keys セクションで新しいキーを作成

### Anthropic API Key
1. https://console.anthropic.com/ にアクセス
2. アカウントを作成/ログイン
3. API Keys で新しいキーを作成

### Google API Key
1. https://makersuite.google.com/app/apikey にアクセス
2. Googleアカウントでログイン
3. 新しいAPIキーを作成

## トラブルシューティング

### Pythonが見つからない
- Pythonがインストールされているか確認してください
- PATHが正しく設定されているか確認してください

### 依存関係のインストールに失敗
- インターネット接続を確認してください
- `pip install -r requirements.txt` を手動で実行してみてください

### 翻訳が動作しない
- APIキーが正しく設定されているか確認してください
- APIキーに十分なクレジットがあるか確認してください
- インターネット接続を確認してください
