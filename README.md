# DeepYami翻訳アプリ

LangChainと複数のLLMモデルを使用した、DeepL風の翻訳アプリケーション

## 機能

- 🌐 **多言語対応**（日本語、中国語簡体字、中国語繁体字、韓国語、英語）
- 🤖 **複数のLLMモデルに対応**
  - OpenAI GPT-4.1
  - OpenAI GPT-4.1-mini
  - Anthropic Claude Sonnet 4.5
  - Anthropic Claude Haiku 4.5
  - Google Gemini 2.5 Pro
  - Google Gemini Flash 2.5
- 🎨 **翻訳スタイル選択**
  - ビジネス：フォーマルで丁寧な表現
  - 同僚：カジュアルプロフェッショナルな表現
  - 友人：フレンドリーでカジュアルな表現
- ✏️ **自動校正機能**：元の言語を維持したまま文法・スペルを修正
- ⚡ **自動翻訳機能**：編集後2秒で自動翻訳（ON/OFF切替可能）
- 🛑 **翻訳・校正のキャンセル機能**：実行中の処理を中断可能
- 💾 **テキスト自動保存**：終了時に自動保存、起動時に自動復元
- 📝 **シンプルなUI**：PanedWindowによる左右2分割レイアウト
- 🖥️ **クロスプラットフォーム対応**：Windows、macOS、Linux

## 必要要件

- Python 3.8以上
- 以下のいずれかのAPIキー：
  - OpenAI API Key
  - Anthropic API Key
  - Google API Key

## セットアップ

### Windows

1. リポジトリをクローンまたはダウンロード
2. `start.bat` をダブルクリック
3. 初回起動時に仮想環境と依存関係が自動的にインストールされます

### macOS / Linux

1. リポジトリをクローンまたはダウンロード
2. ターミナルで `chmod +x start.sh` を実行
3. `./start.sh` を実行
4. 初回起動時に仮想環境と依存関係が自動的にインストールされます

## 使い方

### 初回起動

1. アプリケーションを起動
2. 「設定」メニューから「API設定」を選択
3. 使用したいLLMモデルを選択
4. 対応するAPIキーを入力
5. （オプション）自動翻訳をONにする
6. 「保存」をクリック

### 翻訳

1. 翻訳先の言語を選択（翻訳元は自動検出）
2. 翻訳スタイルを選択（ビジネス/同僚/友人）
3. 左側のテキストエリアに翻訳したいテキストを入力
4. **手動翻訳**: 「翻訳」ボタンをクリック
5. **自動翻訳**: 設定で有効にすると、編集後2秒で自動的に翻訳されます
6. 右側のテキストエリアに翻訳結果が表示されます

### 校正

1. 左側のテキストエリアに校正したいテキストを入力
2. 翻訳スタイルを選択
3. 「校正」ボタンをクリック
4. 元の言語を維持したまま、文法・スペルが修正されます

### その他の機能

- **テキストの自動保存**: アプリ終了時に自動保存され、次回起動時に復元されます
- **コピー**: 翻訳結果の「コピー」ボタンでクリップボードにコピー
- **編集操作**: Ctrl+Z（元に戻す）、Ctrl+Y（やり直し）、Ctrl+A（すべて選択）などに対応

## APIキーの取得方法

### OpenAI API Key
1. https://platform.openai.com/ にアクセス
2. アカウントを作成/ログイン
3. API Keys セクションで新しいキーを作成

### Anthropic API Key
1. https://console.anthropic.com/ にアクセス
2. アカウントを作成/ログイン
3. API Keys で新しいキーを作成

### Google API Key
1. https://makersuite.google.com/app/apikey にアクセス
2. Googleアカウントでログイン
3. 新しいAPIキーを作成

## トラブルシューティング

### Pythonが見つからない
- Pythonがインストールされているか確認してください
- PATHが正しく設定されているか確認してください

### 依存関係のインストールに失敗
- インターネット接続を確認してください
- `pip install -r requirements.txt` を手動で実行してみてください

### 翻訳が動作しない
- APIキーが正しく設定されているか確認してください
- APIキーに十分なクレジットがあるか確認してください
- インターネット接続を確認してください
